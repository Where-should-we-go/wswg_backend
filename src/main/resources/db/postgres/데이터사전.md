# 데이터 사전 (Data Dictionary) — WSWG

**기준**: `backend/src/main/resources/db/postgres/schema.sql`
**작성일**: 2026-06-10
**표기**: PK=기본키 · FK=외래키 · UK=유니크 · NN=NOT NULL

## 🌐 마스터 도메인 (TourAPI 배치 적재)

### users — 회원
| 컬럼 | 타입 | 제약 | 역할 |
|------|------|------|------|
| id | BIGSERIAL | PK | 회원 내부 고유 식별자(자동증가) |
| email | VARCHAR(255) | NN, UK | 이메일. 소셜 로그인 식별·중복가입 방지 키 |
| name | VARCHAR(100) | | 이름/닉네임 (소셜 제공, 없을 수 있음) |
| profile_image_url | TEXT | | 프로필 이미지 URL (소셜 제공) |
| role | VARCHAR(20) | NN, default 'USER', CHECK(USER/ADMIN) | 권한 구분 |
| created_at | DATE | default CURRENT_DATE | 가입일 |

### sidos — 시도
| 컬럼 | 타입 | 제약 | 역할 |
|------|------|------|------|
| sido_code | INTEGER | PK | 시/도 코드 (TourAPI areaCode) |
| sido_name | VARCHAR(100) | NN | 시/도 이름 (서울, 부산…) |

### guguns — 구군
| 컬럼 | 타입 | 제약 | 역할 |
|------|------|------|------|
| sido_code | INTEGER | PK, FK→sidos | 소속 시/도. 복합 PK의 일부 |
| gugun_code | INTEGER | PK, NN | 구/군 코드. **시도 내에서만 유일** → 복합 PK |
| gugun_name | VARCHAR(100) | NN | 구/군 이름 (종로구…) |

### contenttypes — 콘텐츠 타입(=테마)
| 컬럼 | 타입 | 제약 | 역할 |
|------|------|------|------|
| content_type_id | INTEGER | PK | 콘텐츠 타입 코드 (TourAPI contentTypeId) |
| content_type_name | VARCHAR(100) | NN | 타입명 = **테마** (관광지/문화시설/축제/레포츠/숙박/쇼핑/음식점) |

### attractions — 관광지 (핵심 마스터)
| 컬럼 | 타입 | 제약 | 역할 |

| title | VARCHAR(255) | NN | 관광지 이름 |
| content_type_id | INTEGER | FK→contenttypes | 테마 분류 |
| area_code | INTEGER | FK→sidos | 시/도 (= sido_code, TourAPI 명칭) |
| si_gun_gu_code | INTEGER | (복합 FK) | 구/군 (= gugun_code) |
| first_image1 | TEXT | | 대표 이미지 URL |
| first_image2 | TEXT | | 썸네일 이미지 URL |
| map_level | INTEGER | | 지도 확대 레벨 (TourAPI) |
| latitude | DOUBLE | | 위도 (좌표 입력원) |
| longitude | DOUBLE | | 경도 (좌표 입력원) |

| tel | VARCHAR(100) | | 전화번호 |
| addr1 | TEXT | | 기본 주소 |
| addr2 | TEXT | | 상세 주소 |

## 🧳 여행/모임 도메인 (사용자 런타임 생성)

### groups — 모임
| 컬럼 | 타입 | 제약 | 역할 |
|------|------|------|------|
| group_id | BIGSERIAL | PK | 모임 고유 식별자 |
| group_name | VARCHAR(100) | NN | 모임 이름 |
| created_at | TIMESTAMPTZ | NN, default now() | 모임 생성 시각 |

### user_group — 모임 멤버 매핑 (N:M)
| 컬럼 | 타입 | 제약 | 역할 |
|------|------|------|------|
| user_group_id | BIGSERIAL | PK | 매핑 고유 식별자 |
| user_id | BIGINT | NN, FK→users | 멤버(회원) |
| group_id | BIGINT | NN, FK→groups | 소속 모임 |
| joined_at | TIMESTAMPTZ | NN, default now() | 가입 시각 |
| — | | UNIQUE(user_id, group_id) | 같은 모임 중복 가입 방지 |

### trips — 여행
| 컬럼 | 타입 | 제약 | 역할 |
|------|------|------|------|
| trip_id | BIGSERIAL | PK | 여행 고유 식별자 |
| title | VARCHAR(255) | NN | 여행 제목 |
| start_date | DATE | | 시작일 |
| end_date | DATE | | 종료일. **완료 판정: end_date < 오늘** |
| user_id | BIGINT | FK→users, nullable | **개인 여행** 소유자 (XOR) |
| group_id | BIGINT | FK→groups, nullable | **모임 여행** 소유 (XOR) |
| created_at | TIMESTAMPTZ | NN, default now() | 생성 시각 |
| updated_at | TIMESTAMPTZ | NN, **트리거 자동갱신** | 최종 수정 시각 (마이페이지 정렬) |
| — | | CHECK XOR | user_id/group_id 중 **정확히 하나만** |
| — | | CHECK dates | end_date ≥ start_date |

### trip_items — 여행 항목 (유연 데이터 단위, 노션식)
| 컬럼 | 타입 | 제약 | 역할 |
|------|------|------|------|
| item_id | BIGSERIAL | PK | 항목 식별자 |
| trip_id | BIGINT | NN, FK→trips | 어느 여행 |
| content_id | INTEGER | FK→attractions, **nullable** | TourAPI 관광지면 연결, 아니면 NULL(자유 항목) |
| title | VARCHAR(255) | | 직접 입력 제목 (맛집/메모 등). content_id 있으면 생략 가능 |
| item_type | VARCHAR(30) | | 항목 종류 (관광/식당/숙박/이동/메모… 자유) |
| latitude | DOUBLE | | 자체 위치 (관광지 아닐 때) |
| longitude | DOUBLE | | 자체 위치 |
| geom | geometry(Point,4326) | GENERATED, GIST | lat/lng 자동 파생, 지도 뷰용 |
| visit_order | INTEGER | | 방문 순서(일정 뷰 정렬) |
| visit_date | DATE | | 방문 날짜 |
| visit_time | TIME | | 방문 시간 |
| properties | JSONB | NN, default '{}', GIN | **🌟 노션식 자유 필드** (예산/평점/태그/메모…) |
| created_at | TIMESTAMPTZ | NN | 생성 시각 |
| updated_at | TIMESTAMPTZ | NN, 트리거 | 수정 시각 |
| — | | CHECK(content_id 또는 title) | 관광지 링크 없으면 제목은 필수 |

> 같은 데이터를 **일정 뷰**(visit_date/order), **지도 뷰**(geom), **갤러리 뷰**(trip_media)로 다양하게 렌더. 중복 허용.

## 🗺 발자취/미디어 도메인

### trip_region_snapshots — 권역 집계 스냅샷
| 컬럼 | 타입 | 제약 | 역할 |
|------|------|------|------|
| snapshot_id | BIGSERIAL | PK | 스냅샷 행 식별자 |
| trip_id | BIGINT | NN, FK→trips | 어느 여행 |
| sido_code | INTEGER | NN, FK→sidos | 방문한 시/도 |
| gugun_code | INTEGER | nullable | 방문한 구/군 (NULL=구군 미상 허용) |
| attraction_count | INTEGER | NN, default 0 | 그 권역 관광지 수 (**비정규화 집계**, 앱이 갱신) |
| — | | UNIQUE(trip_id, sido_code, COALESCE(gugun_code,-1)) | 권역 중복 방지 |

> 용도: 지도 권역 **색칠(choropleth)** 을 풀스캔 없이 빠르게. 의도적 비정규화 캐시.

### trip_media — 멀티미디어 기록
| 컬럼 | 타입 | 제약 | 역할 |
|------|------|------|------|
| media_id | BIGSERIAL | PK | 미디어 식별자 |
| trip_id | BIGINT | NN, FK→trips | 어느 여행 |
| item_id | BIGINT | FK→trip_items, nullable, **SET NULL** | 어느 항목(장소)에 붙은 미디어. 없으면 여행 전체 |
| user_id | BIGINT | FK→users, **ON DELETE SET NULL** | 업로더 (탈퇴해도 미디어 보존) |
| media_type | VARCHAR(10) | NN, CHECK(PHOTO/AUDIO/VIDEO) | 미디어 종류 |
| media_url | TEXT | NN | 오브젝트 스토리지 경로 |
| geom | geometry(Point,4326) | nullable, GIST | 촬영 좌표(직접 입력) |
| metadata | JSONB | NN, default '{}', GIN | **타입별 유연 메타** (해상도/길이/EXIF/썸네일…) |
| created_at | TIMESTAMPTZ | NN, default now() | 업로드 시각 |

> `metadata` 예시 — PHOTO: `{width,height,thumbnailUrl,exif}` / AUDIO: `{durationSec,format,transcript}` / VIDEO: `{durationSec,codec,resolution,thumbnailUrl}`

## 📌 필드 설계 핵심 메모
- **PK 컨벤션**: 대부분 BIGSERIAL surrogate. 단 sidos/guguns/contenttypes는 TourAPI 코드를 자연키 PK로 사용.
- **관광지 링크 = `attractions.content_id`** (NOT NULL UNIQUE). 단 `trip_items`에선 **선택(nullable)** — 자유 항목 허용.
- **좌표**: lat/lng가 입력원, `geom`은 GENERATED 파생 (3NF, 동기화 불필요). attractions·trip_items 동일 패턴.
- **JSONB 2곳**: `trip_media.metadata`(미디어 메타), `trip_items.properties`(노션식 자유 필드). 둘 다 GIN.
- **trip_items = 유연 데이터 단위**: 하나의 항목이 일정/지도/갤러리 등 여러 뷰의 원본. 미디어가 붙으면 계획→기록 자연 전환.
- **시각 타입**: TIMESTAMPTZ (단 users.created_at만 DATE — 현행 유지).
